## üìà Key Features: Beyond Basic Plotting

This dashboard is designed specifically for time-series industrial data analysis, solving the tedious problem of manual data formatting and chart creation.

### 1. Dynamic Axis Selection
* **Function:** Automatically populates X/Y dropdown menus with all column headers from the loaded CSV file.
* **Value:** Allows users to quickly compare any two variables (e.g., Temperature vs. Time, or RPM vs. Pressure).

### 2. Time-Series Optimization
* **Function:** Automatically parses `YYYY-MM-DD HH:MM:SS` timestamps and formats the X-axis for professional, readable time scales.
* **Value:** Eliminates the problem of cluttered or misaligned time values common when plotting large datasets.

### 3. Core Framework
* **Toolkit:** Built with Tkinter (GUI), Pandas (Data Handling), and Matplotlib (Visualization).
* **Usability:** Simple "Load File" interface for instant analysis.
